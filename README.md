## 功能介绍（完整见运行截图）
管理员：管理员通过登录界面进行身份验证，进入系统后可访问多个管理模块。主要功能包括修改密码、管理部门信息、员工档案录入与管理、查看和发布通知公告、管理请假信息、管理薪资信息及考勤记录。管理员可以添加新员工或更新员工资料，并有权限删除员工信息。此外，还可以进行部门结构管理，通过树形图操作轻松增删部门，提高日常管理的效率和便捷性。
